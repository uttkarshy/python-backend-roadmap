# Visitor Registration System

## Description

A console-based Visitor Registration System built using Python Sets.

This project demonstrates how Sets automatically prevent duplicate entries while allowing users to register, search, remove, and count visitors.

## Features

- Register Visitor
- View Registered Visitors
- Search Visitor
- Remove Visitor
- Count Visitors
- Exit Program

## Concepts Used

- Python Sets
- Loops
- Conditional Statements
- Functions (optional in future versions)
- Exception Handling

## Future Improvements
