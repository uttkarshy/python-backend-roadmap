# 06 - Sets and Tuples

## Overview

This folder contains my practice work for Python Sets and Tuples.

## What I Learned

### Sets
- Sets store unique values
- Duplicate values are ignored
- Sets are useful when order is not important
- Common operations:
  - add()
  - remove()
  - len()
  - loop through values

### Tuples
- Tuples store fixed values
- Tuples cannot be changed after creation
- Tuples are useful for data that should not change
- Common operations:
  - indexing
  - len()
  - loop through values
